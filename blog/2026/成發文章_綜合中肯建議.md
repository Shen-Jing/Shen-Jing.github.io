# 一、先還原你是什麼樣的寫作者

讀了你 2009 到 2026、橫跨十七年的文章後，我看到的你是這樣一位寫作者：

- **誠實到願意自曝其短**：拉鍊沒拉、上台「一坨糨糊」、便祕當預兆、為《魔物獵人》組電腦、夢到被資遣醒來還很開心——你不美化自己，這是最難得的底氣。
- **高度自省、以「了解自己」為人生主軸**：你反覆把「了解自己」當成目標，用閱讀、冥想、寫作當工具。你甚至會回頭評點 13 歲的〈夜車〉、16 歲的《追風箏的孩子》心得，這種對自身成長的後設覺察，很多人一輩子都沒有。
- **建構型的思考者**：你不寫單篇，你在織網。斯多葛、心流、餘裕、非暴力溝通、動態眼光……這些概念在你筆下互相支援，形成一套自洽的生活哲學。
- **重情、念舊、感恩**：從阿公的十元硬幣、阿嬤的「呷飽未」，到一一向老師同學道謝——你的文章底色始終是溫的。

這是很好的地基。所以下面的「成長邊」，不是要你變成另一個人，而是幫你**把已經很好的東西，再推一階**。

# 二、你已經做得很好的（請繼續）

1. **持續的書寫習慣**：你把寫作當成晨間 routine 與「沉澱思緒」的儀式，這份穩定本身就是稀缺資源。
2. **具體的生活細節**：你很擅長用一個小物件、一個動作承載情感（硬幣、拉鍊、左手扶尺）。這是你最強的肌肉。
3. **概念與生活的雙向印證**：你不是空談理論，而是讓塞內卡去解釋你和老婆的真實事件——這讓抽象概念有了體溫。
4. **對讀者的善意**：註腳、延伸閱讀、推坑書單，看得出你想「為周遭的人提供新視角」。這份心意讀者感受得到。

# 三、中肯的成長邊（附跨篇證據）

> 這部分我盡量直話直說，但每一點都來自你文章裡反覆出現的模式，不是單篇的吹毛求疵。

## 3-1. 「解釋」多於「呈現」——而你自己早就診斷出來了

你在《追風箏的孩子》的「2026 回顧」親手寫下：當年的自己「**結構不夠完整，感覺都只是點到為止**」「**很線性地圍繞在故事的情節上**」；在〈夜車〉也自評「**轉折處的鋪陳稍嫌倉促**」。

十幾年後，這個傾向其實還在，只是換了形式：**你很快地把情緒「講出來」（最打動我、很尷尬、超快樂、鼻酸），卻較少把讀者「放進那個當下」讓他們自己感受到。** p.md 的「弟弟轉頭」「忘記翻譜」都是如此。

→ **建議**：對最重要的兩三個瞬間，用「慢鏡頭＋身體反應＋當下念頭」去演出來，而不是總結。（〈寫作潤飾建議〉那篇有三段示範可直接參考。）

## 3-2. 思緒容易發散，收束偏弱

這是你最一致的模式。證據俯拾即是：

- 〈對生產力…〉：「話題似乎稍微扯遠了」「礙於篇幅以及逼近的死線，這次先梳理到這」。
- 〈理想的日常〉：自承「拖稿」，因為「思緒越挖越深」。
- 多篇結尾都是「還有太多面向值得探討」式的開放收尾。

你的腦子是「聯想引擎」，這是優點（所以你能織網），但代價是**單篇常常裝太多、又收不攏**。p.md 的「綜合心得」段就是縮影：聽別人演出、兄弟、自己練琴插曲、拉鍊，全擠在一起。

→ **建議**：替每篇定**一個核心問句**，寫的時候反覆自問「這段有沒有在回答它？」沒有的就移到註腳或下一篇。**「少寫一點，但說透一點」**，對你會是最大的升級。

## 3-3. 概念引用，有時成了「情感的防彈衣」

你博覽群書，隨手能召喚斯多葛、心流、冒牌者症候群、非暴力溝通、False Hope Syndrome……這是你的魅力，但也藏著一個風險：

**當情緒一上來，你常常很快地切換到「概念解釋模式」，用一個學名把感受收編、安放好。** 這讓文章顯得睿智，卻有時也讓讀者（甚至你自己）少了一次「純粹地待在那個感受裡」的機會。比如喪親那篇，你用「冒牌者心態」去命名「懷疑自己有沒有資格緬懷」——精準，但那份痛，其實值得你先不貼標籤地多寫幾句。

→ **建議**：試試「**先血肉、後概念**」。先把赤裸的經驗與情緒寫到底，最後再（如果需要）請出那個概念來收尾，而不是讓概念太早進場「維持秩序」。

## 3-4. 「過度準備」是你的主題，也是你的寫作慣性

你在〈不想只謝天呀〉坦承內耗常源於「習慣先做準備」；成發前你錄影錄到自我厭惡、追求零失誤。這份「過度準備／完美傾向」，某種程度也反映在寫作上——**想把所有面向都顧到、所有概念都接上，於是篇幅膨脹、遲遲收不了尾、拖稿**。

→ **建議**：把你對自己很受用的「**自我慈悲**」和「**控制二分法**」也用在寫作上。一篇文章不必窮盡一個主題（那不歸你控制），它只要**誠實地完成一次思考**就夠了。允許自己交出「不完美但完整」的文章——這恰好就是你成發學到的那課：「我不太適合追求完美」。

## 3-5. 情感最濃處，符號反而稀釋了力道

emoji 與「QQ／🥹／🫠」是你的招牌語氣，平時很可愛、能調節節奏，請保留。但在**真正動人的那一兩句**（弟弟轉頭、老師說「我們」、留下活著的痕跡），符號會把情緒「講白」，讀者就不必自己去感受了。

→ **建議**：分清楚「日常語氣段」和「情感高點段」。前者照常賣萌，後者**乾淨、留白、不放表情符號**。讓最重的話，自己站著。

# 四、給你的四個寫作實驗（可挑著玩）

1. **一篇一問**：下篇開寫前，先用一句問句寫在最上面（例如：「為什麼一場業餘成發，會讓我想起弟弟？」），全篇只服務這個問句。
2. **場景先行**：強迫自己用一個「畫面」開場（人、地、動作、一句對話），把概念全部延到後半段才登場。
3. **減引用挑戰**：選一篇，刻意把書名、學名、概念詞砍掉一半，看看光靠你自己的觀察與比喻，能不能把同一件事說得一樣好。（你絕對可以，你的細節力很強。）
4. **「寫給弟弟的一封信」**：p.md 挖出了一塊金礦——你和弟弟「沒能一起完成一件事」的悵惘（並與你 17 歲寫的《追風箏的孩子》遙遙呼應）。試著單獨為它寫一篇，甚至真的拿去邀他一起做點什麼。這可能是你近期最值得寫、也最值得「活出來」的題目。

# 五、關於 p.md 本身：最終定位建議

這是一篇**水準之上、值得發表**的文章，不要因為上面一堆建議就覺得它不好。它只需要「**收斂 + 深化**」，不需要重寫。具體三步：

1. **選一條主脊**：建議從「**被看見／與不完美的自己和解**」或「**心流／在無常裡留下活著的痕跡**」二選一，作為全篇的脊椎，其餘素材圍著它排。
2. **深化三個點**：弟弟轉頭、忘記翻譜、老師那句「我們」——用慢鏡頭重寫（見〈寫作潤飾建議〉示範）。
3. **拉兩三條線**：往《心流》（為什麼開始）、往手足悵惘、往「被看見」，各補一句回扣（見〈跨篇思想連結與深化〉工具箱）。

做完這三步，**其餘的就放手**。別讓「想顧到全部」又把它變成一篇拖稿的長文。

# 六、最後，一句中肯也由衷的話

你 13 歲在〈夜車〉的結尾寫：「我要振作起來，渾渾噩噩地過著日子也不是辦法！」；16 歲在《追風箏的孩子》結尾寫下「為你，千千萬萬遍」；如今 30 多歲的你，在失去三位至親的那一年，仍願意坐到鋼琴前、坐到書桌前，「留下點活著的痕跡」。

**你最大的天賦不是文筆，是『願意誠實地活著、並且把它記下來』這件事本身。** 文筆是可以練的——少裝一點、慢一點、留白多一點，你早晚會到。但那份誠實與溫度，是很多技巧再好的人都換不到的。

所以我的綜合建議只有一句：

> **繼續寫。寫少一點、寫深一點、對自己寬一點。** 你已經在對的路上，只是還沒給自己足夠的肯定——而這，正好也是你這場成發學到的事。
